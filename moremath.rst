More Math
=========

There are lots of ways we can combine numbers or modify them.
We're going to call one of our numbers `count` here just to 
make it easier to see what the statements do:

.. doctest::

    >>> count = 5.0
    >>> count * 3
    15.0
    >>> count / 3
    1.6666666666666667
    >>> count ** 2 # exponentiation
    25.0
    >>> count ** .5
    2.23606797749979
    >>> count % 3 # remainder
    2.0
    >>> count // 3 # quotient
    1.0
    >>> count * 3 + 2
    17.0
    >>> count * (3 + 2)
    25.0
