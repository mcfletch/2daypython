Where are You?
==============

Try to answer the following puzzles:

* what does the following do?

  .. code-block:: python
  
    while True:
        print( hello world )

* what does the following do?

  .. code-block:: python
  
    while True:
        print("Hello World")

* what does the following do?

  .. code-block:: python
  
    x = 10
    while True:
        print(x)
        x = x - 1

* what does the following do?

  .. code-block:: python
  
    x = 5
    while True:
        if x > 3:
            print(x)
        x = x - 1

* what does the following do?

  .. code-block:: python
  
    x = 3
    while x > 3:
        print(x)
        x = x - 1

* what does the following do?

  .. code-block:: python
  
    x = 3
    while x < 3:
        print(x)

* what does the following do?

  .. code-block:: python
  
    x = "this"
    while x:
        print(x)
        x = x[:-1]

* what does the following do?

  .. code-block:: python
  
    x = (2,3)
    while x[0] < 5:
        x = x[0] + 1, x[1]

* what does the following do?

  .. code-block:: python

    x = 10
    while x > 5:
        if x <= 7:
            x = x + 5
        else:
            x = x - 3
        print(x)
    
* what do these directions mean in Pygame (left, right, up, down, how fast)?

    * (-1,1)
    * (-10,20)
    * (0,10)
    * (10,1)
