Comparisons
===========

* Remember we need to create `tests` that decide whether to do something
* The result of a `test` is a `boolean` `True` or `False` value
* We often use `comparisons` to produce those `True` or `False` values

.. doctest::

    >>> 8 == 8
    True
    >>> 8 == 9
    False
    >>> 8 != 8
    False
    >>> 2 > 3
    False
    >>> 0 < 4
    True
    >>> 3 >= 3
    True
    >>> 4 <= 6
    True
    >>> True == False
    False
    >>> True == 1
    True 
    >>> False == 0
    True
    >>> 1 == 1.0
    True
